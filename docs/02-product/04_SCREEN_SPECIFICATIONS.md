# NOURA Screen Specifications

Version: 1.0

Status: Living Document

---

# Purpose

Screen Specifications define the responsibility of every screen inside NOURA.

They ensure every screen has:

- One clear purpose.
- The right information.
- The right actions.
- Consistent behavior.

Design should never begin before the screen specification is complete.

---

# Screen Template

Every screen must define:

- Purpose
- User Goal
- Primary Question
- Displayed Information
- Primary Actions
- Secondary Actions
- Navigation
- States
- Success Criteria

---
