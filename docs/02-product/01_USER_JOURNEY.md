Version: 1.0

# Purpose

NOURA is designed to help users achieve Financial Clarity.

Financial Clarity is not created in a single interaction.

It is built through a continuous journey.

This document describes how users evolve while using NOURA.

Every feature exists to support this journey.

---

# The Journey

Unknown

↓

Aware

↓

Organized

↓

Understand

↓

Confident

↓

Peace of Mind

---

# Stage 1

## Unknown

### User Mindset

"I don't really know where my money goes."

### Characteristics

- Never records finances consistently
- Often forgets previous expenses
- Relies on memory
- Unsure of current financial condition

### User Questions

How much money do I actually have?

Where did it go?

Can I improve my financial habits?

### NOURA Responsibility

Introduce a simple and trustworthy financial system.

### Success

The user is willing to start.

---

# Stage 2

## Aware

### User Mindset

"I've started recording my finances."

### Characteristics

- Financial Setup completed
- Records transactions regularly
- Builds financial history

### User Questions

Am I recording everything?

Can I trust these numbers?

### NOURA Responsibility

Build trust through consistency.

### Success

The user trusts NOURA as the source of truth.

---

# Stage 3

## Organized

### User Mindset

"My finances are now organized."

### Characteristics

- Transactions categorized
- History is complete
- Financial data is structured

### User Questions

What does all this data mean?

### NOURA Responsibility

Transform records into understandable information.

### Success

The user has a complete financial picture.

---

# Stage 4

## Understand

### User Mindset

"I understand my financial condition."

### Characteristics

- Understands income
- Understands expenses
- Understands spending patterns

### User Questions

What should I improve?

Where can I save money?

### NOURA Responsibility

Generate meaningful insights.

### Success

The user understands both their current condition and historical patterns.

---

# Stage 5

## Confident

### User Mindset

"I know what financial decisions to make."

### Characteristics

- Makes informed decisions
- Builds healthy financial habits
- Feels in control

### User Questions

How can I achieve my goals faster?

### NOURA Responsibility

Provide guidance and recommendations.

### Success

The user manages money with confidence.

---

# Stage 6

## Peace of Mind

### User Mindset

"I no longer worry about my finances."

### Characteristics

- Financial information is always available
- Financial condition is always understandable
- Decisions are no longer based on guesswork

### NOURA Responsibility

Maintain long-term Financial Clarity.

### Success

Financial management becomes effortless.

---

# Journey Principles

Financial Clarity grows gradually.

↓

Recording creates history.

↓

History creates understanding.

↓

Understanding creates confidence.

↓

Confidence creates peace of mind.

# Success Definition

Users should not feel that they are using a finance application.

They should feel that they always understand their financial condition.

That understanding is Financial Clarity.

Financial Clarity leads to Financial Confidence.

Financial Confidence leads to Peace of Mind.
