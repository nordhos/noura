# NOURA Ubiquitous Language

Version: 1.0

Status: Living Document

---

# Purpose

Every important term used in NOURA must have one shared meaning.

Designers.

Developers.

Product.

Documentation.

Marketing.

Everyone should use the same language.

Consistency in language creates consistency in products.

---

# Financial Clarity

Definition

A user's ability to understand their financial condition quickly, accurately and confidently.

This is the primary outcome NOURA delivers.

---

# Financial System

Definition

The complete financial structure created during Financial Setup.

It consists of:

- Financial Timeline
- Profile(s)
- Categories
- Transactions

The Financial System becomes the foundation of every future calculation.

---

# Financial Setup

Definition

The one-time process of creating a user's Financial System.

Financial Setup is not account registration.

Financial Setup establishes the foundation for Financial Clarity.

---

# Financial Timeline

Definition

The date from which NOURA begins calculating the user's financial history.

Every transaction must exist on or after this date.

---

# Profile

Definition

A financial owner.

Every transaction belongs to exactly one profile.

A profile is not a login account.

A profile represents a person whose finances are being managed.

Maximum profiles:

2

---

# Initial Balance

Definition

The amount of money owned by a profile at the beginning of the Financial Timeline.

Implementation

Represented as an Income transaction.

Category:

Saldo Awal

---

# Transaction

Definition

A financial event.

Every income.

Every expense.

Every balance calculation originates from transactions.

Transactions are immutable historical records.

---

# Category

Definition

A classification used to organize financial transactions.

Categories improve understanding.

Categories do not affect calculations.

---

# Balance

Definition

The calculated result of:

Total Income

minus

Total Expense

Balance is never stored.

Balance is always calculated.

---

# Dashboard

Definition

The primary place where users understand their current financial condition.

Dashboard is not a reporting page.

Dashboard answers:

"How am I doing today?"

---

# Report

Definition

A structured summary of financial history.

Reports explain financial progress over time.

---

# Financial Health

Definition

A high-level assessment of financial condition based on historical financial activities.

Financial Health is an interpretation.

Not a transaction.

Not a balance.

---

# Insight

Definition

An explanation generated from financial data.

Insights help users understand.

Not simply observe.

---

# Financial Confidence

Definition

The confidence users gain after achieving Financial Clarity.

Financial Confidence is the desired long-term outcome of NOURA.

---

# Peace of Mind

Definition

The emotional outcome produced by Financial Clarity.

It is the ultimate experience NOURA aims to deliver.
